## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/malte-moehring/sundspace-website.git
    ```
2. Navigate to the project directory:
    ```bash
    cd sundspace-website
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```

## Usage
